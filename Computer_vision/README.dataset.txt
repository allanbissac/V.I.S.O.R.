# BlockClassifier > ObjectClassification_v5

Provided by a Roboflow user

The objective of this project is to make a robot accurately detect and classify differently coloured blocks scattered in an environment. What the robot should do is to identify target blocks, pick them and place the block inside a bin of matching colour (e.g. red block inside red bin).
